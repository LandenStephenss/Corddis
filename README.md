# Corddis
A really cool Discord library

## Installing
You will need Node 12.13+. To install Node.js, click [here](https://nodejs.org).
* Clone: `git clone --branch dev https://github.com/Apacheopteryx/Corddis`
* Install dependencies: `npm i`
